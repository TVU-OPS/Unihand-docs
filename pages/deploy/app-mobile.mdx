# 📱 Hướng dẫn Build Ứng dụng React Native với Expo

Thư mục `unihand/app-mobile` chứa mã nguồn ứng dụng di động React Native. Làm theo hướng dẫn dưới đây để build ứng dụng bằng **Expo**.

## 🛠️ Yêu cầu hệ thống

Trước khi bắt đầu, đảm bảo rằng bạn đã cài đặt các công cụ và phần mềm cần thiết:

- **Node.js**: Phiên bản LTS (tối thiểu 14.x).
- **npm** hoặc **yarn**: Trình quản lý gói.
- **Expo CLI**: Công cụ dòng lệnh của Expo.

Cài đặt Expo CLI nếu bạn chưa có:
```bash
npm install -g expo-cli
```

## 🔧 Cài đặt dự án

1. **Clone dự án**:
   ```bash
   git clone https://github.com/TVU-OPS/UniHand.git
   ```

2. **Di chuyển vào thư mục `app-mobile`**:
   ```bash
   cd UniHand/app-mobile
   ```

3. **Cài đặt các phụ thuộc**:
   ```bash
   npm install
   # Hoặc
   yarn install
   ```

## ▶️ Chạy ứng dụng bằng Expo

1. **Khởi động ứng dụng**:
   ```bash
   expo start
   ```

2. **Chọn cách chạy ứng dụng**:
   - Mở **Expo Go** trên điện thoại của bạn và quét mã QR hiển thị trên terminal hoặc trình duyệt.
   - Nếu sử dụng máy ảo:
     - Chọn `Run on Android device/emulator` để chạy trên máy ảo Android.
     - Chọn `Run on iOS simulator` (chỉ dành cho macOS) để chạy trên trình giả lập iOS.

## 🏗️ Build file APK/IPA bằng Expo

Expo hỗ trợ build ứng dụng trực tiếp trên máy chủ của họ.


1. **Cài đặt Expo CLI**
    ```bash
    npm install -g expo-cli
    ```

2. **Đăng nhập vào tài khoản Expo**:
   ```bash
   eas login
   ```
   Nếu chưa có tài khoản, bạn cần [đăng ký](https://expo.dev/) tại Expo Developer Portal.

3. **Build file APK (Android)**:
   ```bash
   eas build -p android --profile preview
   ```

4. **Tải xuống file build**:
   Sau khi build xong, Expo sẽ cung cấp link để tải file APK/IPA.
   - B1: Đăng nhập [Expo](https://expo.dev/)
   - B2: Vào project

   - B3: Chọn tập tin UniHand vừa buid
      # ![UniHand Banner](/img/app_mobile_build_b2.png)
   - B4: Tải xuống
      # ![UniHand Banner](/img/app_mobile_build_b3.png)

## 🐛 Gỡ lỗi

- Nếu gặp lỗi khi chạy ứng dụng, sử dụng lệnh sau để kiểm tra log:
  ```bash
  expo diagnostics
  ```

- Đảm bảo rằng:
  - Máy ảo hoặc thiết bị thật đã được kết nối đúng cách.
  - Các biến môi trường trong file `.env` đã được cấu hình chính xác.
